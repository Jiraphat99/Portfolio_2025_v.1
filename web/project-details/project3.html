AI Business Ideas Generator

Overview

This AI-powered web app generates unique business ideas using the SheCodes AI API. Users can input keywords or themes, and the app produces creative, practical business concepts instantly.


My Role: Frontend Developer


Tools Used: HTML · CSS · JavaScript · SheCodes AI API

Highlights

  • Uses SheCodes AI API to generate business ideas in real-time.
  • Users can input keywords to get customized results.
  • Displays ideas in a clean, responsive card layout.
  • Fast and interactive user experience.
  • Helps entrepreneurs brainstorm unique business concepts quickly.

Tech Stack

  • HTML5 for semantic structure
  • CSS3 for responsive layout and styling
  • Vanilla JavaScript for API integration and DOM updates
  • SheCodes AI API for business idea generation
  • Netlify for hosting and deployment
