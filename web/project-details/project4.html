AI Thai Poem Generator

Overview

The AI Thai Poem Generator uses the SheCodes AI API to create original Thai poems based on user-provided keywords or themes. It helps users explore creative writing and generate beautiful, culturally rich content instantly.


My Role: Frontend Developer


Tools Used: HTML · CSS · JavaScript · SheCodes AI API

Highlights

  • Generates Thai poems in real-time using the SheCodes AI API.
  • Users can input keywords or themes to customize poems.
  • Displays generated poems in a clean, readable layout.
  • Fast, interactive, and responsive interface across devices.
  • Encourages creativity and exploration of Thai literature.

Tech Stack

  • HTML5 for semantic structure
  • CSS3 for responsive layout and styling
  • Vanilla JavaScript for DOM manipulation and API integration
  • SheCodes AI API for poem generation
  • Netlify for hosting and deployment
